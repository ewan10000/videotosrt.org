# VideoToSRT

VideoToSRT is an online subtitle editor and AI transcription tool for creating, editing, translating, and exporting subtitles in the browser.

Website: [https://videotosrt.org](https://videotosrt.org)

## Project

This repository contains the Next.js App Router implementation for VideoToSRT. It uses TypeScript, Tailwind CSS, OpenNext, and Cloudflare Workers.

## Development

```bash
npm install
npm run build
npm run preview
```

Deploy only after Wrangler is authenticated:

```bash
npm run deploy
```
